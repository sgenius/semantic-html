Common non-semantic tags

A generic table with non-semantic tags
Name Usage
table and its children create tables, like this one
dl and children For definition lists; see below
i, b For italics and bold, but do not use this; prefer the tags that mean why the text has to be italic or bold, such as em (emphasis) or mark (highlight due to relevance).
div A generic block. Use if no better alternative exists.
span A generic text. Use if no better alternative exists.

Some definitions

Mexico
The name of a country, and also a state in that country, and also the capital of that country that is nonetheless not in the state of the same name.
Element
In HTML, a component of the page. It's represented by a tag.

A note on "role"

The attribute "role" provides a role to elements that do not have one.

For example, this is a mathematical expression: 2 + 3 = 5.

Note: The ARIA role "note" can provide an aside when other tags do not make sense.
